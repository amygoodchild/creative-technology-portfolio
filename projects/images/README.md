# Project Images

Each project has its own subfolder for organizing images:

```
images/
├── project-1/          # Healthcare Appointment System
│   ├── hero.jpg
│   ├── research.jpg
│   ├── prototypes.jpg
│   └── final-screens.jpg
├── project-2/          # Organic Flow Fields
│   ├── hero.jpg
│   ├── exploration.jpg
│   ├── flow-field.jpg
│   └── variations.jpg
├── project-3/
└── ...
```

## Adding Images

1. Create a folder for your project if it doesn't exist
2. Add your images to that folder
3. Update the HTML to reference them:

```html
<!-- Replace placeholder divs with: -->
<img src="images/project-1/hero.jpg" alt="Description of image">
```

## Recommended Sizes

- **Hero images:** 1800x1200px (16:9 ratio)
- **Content images:** 1800x1125px (16:10 ratio)
- **Project card thumbnails:** 1600x1000px (16:10 ratio)

Format: JPG or PNG, optimized for web (under 500KB each)
