Handwriting

I recreated my handwriting in JavaScript, using only a handful of defined points for each letter. The points are passed through a couple of algorithms to turn them into natural, variable width wordforms with variety and ligatures.
JavaScript
p5.js
plotting

Defining the letters

Each letter is defined by a small number points, which means the file size for the handwriting code is relatively small and defining new letters is quick. The points are curved into smooth paths using Chaikin's curve algorithm and then turned into a shape so that they can have a varied width.
Letter paths
Curve algorithm on a simple path
I created a small tool to define letters quickly, by clicking out the paths and then exporting the point positions as an array. In this tool I could also upload a photograph sample of my handwriting to trace.
